### 源码安装

```sh
go get -d -u github.com/xuqingfeng/mailman

cd $GOPATH/src/github.com/xuqingfeng/mailman

make run
```

- [Hotmail](https://www.hotmail.com/)
- [Outlook](https://www.outlook.com/)
